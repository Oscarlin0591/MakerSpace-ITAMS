# Makerspace Inventory Tracking and Management System (ITAMS)

## Overview

The Makerspace ITAMS project is a full-stack, automated nventory system designed to track consumables within the Quinnipiac University Maker Space. It will eliminate manual inventory checks by utilizing a camera and AI integration and ensure users are notified when material are low.

## Features

Live Dashboard
Data Visualization
Low-stock Alerts
